# Node Chat App

## Introduction

## Usage

### Installing the Application

```bash
cd www
npm install
cd ..
cp sample_env .env
docker-compose build
```

### Run the Application

```bash
docker-compose up -d
```

### Close the Application

```bash
docker-compose down
```
